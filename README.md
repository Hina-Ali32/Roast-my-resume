# 🔥 Roast My Resume

A web app that gives brutal honest AI feedback on your resume.

## Live Demo
[Click here to see it live](#) ← we will add the link after deployment

## What It Does
Paste your resume or upload a .txt file and get honest, detailed AI feedback instantly.

## Built With
- React
- Tailwind CSS
- Claude AI API
- Netlify Functions

## Features
- Paste resume text directly
- Upload resume as a file
- AI powered feedback
- Fully responsive on mobile and laptop
- Secure API key handling with Netlify Functions

## How To Run Locally
1. Clone the repository
2. Run `npm install`
3. Add your Claude API key in `.env` file
4. Run `npm start`
